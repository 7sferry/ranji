# ranji

A free, private, browser-based family tree builder. Create and visualize your genealogy — add relatives, map
parent/child and spouse relationships, switch between an interactive tree canvas and a list view, and import/export your
data as JSON.

**Everything runs client-side.** No accounts, no servers, no tracking. All trees are stored in your browser's local
storage — clearing your browser data erases them, so use **Export** for backups.

Live at [treefam.netlify.app](https://treefam.netlify.app/).

Built with React 19, Vite, TypeScript, and Tailwind CSS v4.

## Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run dev
```

This project was created using bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
